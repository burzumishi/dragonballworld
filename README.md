Dragon Ball World
=================

*"Dragon Ball World"* is an Online Fight Game.

The source code is a fork from *ZEQ2-Lite (rev1536)* with merged *ioq3 (v1.36) bots*.

The goal of this project is to play *Dragon Ball* worlds online:

  * _Dragon Ball_
  * _Dragon Ball Z_
  * _Dragon Ball GT_
